Average Squares documentation
=============================

This project provides utilities for computing weighted averages of squared values
and converting string representations of numbers to numerical lists.

The main functionality includes:

* Computing weighted average of squares with optional custom weights
* Converting strings containing numbers (with whitespace) to float lists

This documentation will guide you through the usage and implementation details
of the Average Squares module.
